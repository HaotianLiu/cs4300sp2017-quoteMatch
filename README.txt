Project.ipynb: implementation of cosine-based IR system, Topic Modeling with stemming and KL-divergence as similarity measure, also introduces application of "fake" documents that mitigates the problem of short documents.

fake_documents.json: used to create fake documents. Created by expand each word in the original document using Stanford Empath.

contractions.json: common English contractions used to expand contractions and later used for stemming

quotes.csv:data

corpora: contains wordnet data for stemming
